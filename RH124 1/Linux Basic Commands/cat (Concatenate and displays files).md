Displays file content, merges files.

·      Common Options:

·      **cat file** **→** Show content

·      **cat file1 file2** → Merge files

·      **cat > file** → Create file (overwrite)

·      **cat >> file** → Append to file

·      **cat -n file** → Show with line numbers
